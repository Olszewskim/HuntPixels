# HuntPixels


Podgląd rozgrywki na której moglibyśmy bazować. Podzieliłem go na 3 etapy:
1. Na lewym screenshocie widać początek "planszy":
* 		ile kafelków danego koloru potrzebujesz zebrać
* 		ile masz ruchów
* 		szare pole do "malowania"
* 		na dole plansza z kolorowymi kafelkami
 2. Środkowy screenshot pokazuje mechanikę zbierania kolorów, która jest prosta. Musisz przytrzymać palcem na początkowym kafelku, który Cię   interesuje i przeciągasz nim (palcem) po kafelkach tego samego koloru (jeśli są obok siebie w pionie/poziomie/po skosie) im większego chaina złapiesz tym więcej "koloru" otrzymasz, który automatycznie wypełni szarą planszę.
Na na miejscu zebranych kafelków wygenerują się nowe
 3. Prawy screenshot pokazuje koniec poziomu - czyli pomalowaną planszę
